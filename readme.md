Raspberry Pirate Radio
====

A set of tools to non-stop-stream / FM broadcast music.

Features
====
* Start automaticaly
* Autonomous (work without network)
* Monitored/admined on the network when available
* Automatic music download


Requirements
====
* Rpi Model B
* USB FM Transmitter
* Python
* Php ?
* Mysql ?
* XBMC distrib
